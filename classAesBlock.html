Detailed Description

Class representing a single 128-bit block of AES encryption.

Provides the core transformations (SubBytes, ShiftRows, MixColumns, AddRoundKey, KeyExpansion) necessary for AES-128 encryption. This implementation focuses on simplicity and correctness rather than maximum performance, to be used as a building block for sequential and parallel modes of operation.
